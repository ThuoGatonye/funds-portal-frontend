Roles Management

Defined Roles

Admin

Description: Full administrative access to the system, managing users, settings, and all reports.

  • User Management (CRUD)
  • Group Management (CRUD)
  • All Reports Access
  • System Settings
  • Audit Logs View

Treasurer

Description: Manages financial aspects, contributions, payouts, and related reports.

  • Record Contributions
  • Manage Payouts
  • Financial Reports Access
  • No User Management

Member

Description: Standard member access to view their own profile and contributions.

  • View Own Profile
  • View Own Contributions
  • No Administrative Access

More role details or a DataTables if roles become numerous.
